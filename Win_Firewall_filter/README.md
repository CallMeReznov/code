# Win_Firewall_filter


## 干嘛用的？

之前在游戏公司工作时，遇到网络攻时顺手写的防火墙过滤脚本,主要过滤外挂与攻击者攻击接口<br>
本来是想全部使用PS调用.NET来处理IIS生成的日志，但LogParser的处理速度实在是太优秀了，还支持SQL语句<br>
测试了PS自带与调用.NET来处理都不如他来的快速
